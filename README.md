### Prerequisites

- Flutter SDK (>=3.0.0)
- Dart SDK (>=3.0.0)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/tilakpatel22/employee-attendance-app-flutter.git
   cd employee-attendance-app-flutter
   ```

   ```bash
   flutter pub get
   ```
